# Decomposition of Pets Shelter App from a monolithic application into independent microservices that work interoperably

1. Plan a microservices application for the Pets Shelter App, and develop the frontend for the App.
2. Built the first microservice for the Pet Shelter App, called GetPets microservice.
3. Built the Applications microservice--provides the functionality to allow Pet Shelter employees to pull detailed information about applications for adoption in the Pet Shelter App.
4. Built a microservice that allows a prospective pet adopter to apply online for pets at the Pet Shelter, the PushPets microservice.
5. Ensured the security of the Pet Shelter App by creating authentication and authorization microservices using Amazon Cognito.
6. Added a Reporting microservice by configuring Amazon SNS and Step Functions with SAM to deliver email reports from the Pet Shelter App (to different audiences).
7. Used CloudWatch and AWS X-Ray to monitor resources , collect metrics, and analyze the behavior of the PetShelter App (Observability and Tracing)
